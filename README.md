# ZhmBucket 服务器文档网站

## 项目简介
这是 ZhmBucket Minecraft 服务器的官方文档网站，提供服务器信息、游戏规则、命令列表、常见问题解答等内容，简约美观。你可以对其进行修改对你的服务器。

## 可修改内容说明

### 中文
以下是您可以根据需要自由修改的内容：

1. **服务器信息**
   - 服务器名称、版本、IP地址等基本信息
   - 服务器介绍文本和描述

2. **游戏规则与命令**
   - 游戏规则列表和说明
   - 命令列表及其功能描述
   - 可添加新的命令项或删除不需要的命令

3. **常见问题**
   - 问题列表和解答内容
   - 可根据服务器实际情况更新

4. **新闻公告**
   - 最新公告内容和发布日期
   - 可定期更新服务器活动信息

5. **服务器截图**
   - 替换为您自己的服务器截图
   - 保持文件命名一致或更新HTML中的引用

### English
The folder 'en' in the source code provides you with the English version of the HTML.
Below are the contents you can freely modify according to your needs:

1. **Server Information**
   - Server name, version, IP address and other basic information
   - Server introduction text and description

2. **Game Rules and Commands**
   - List of game rules and descriptions
   - Command list and function descriptions
   - New command items can be added or unnecessary commands can be removed

3. **Frequently Asked Questions**
   - Question list and answer content
   - Can be updated according to the actual situation of the server

4. **News and Announcements**
   - Latest announcement content and release date
   - Server activity information can be updated regularly

5. **Server Screenshots**
   - Replace with your own server screenshots
   - Keep file naming consistent or update reference paths in HTML

6. **Server Map**
   - Update the map iframe URL to your own map service address
   - Adjust map description text

## 开发说明
- 修改内容时请保持HTML结构的完整性
- 如需更改样式，请编辑css/style.css文件
- 响应式设计已配置，请确保在修改时保持其兼容性
- 英文版源代码副本位于/en目录中

## 部署方法
1. 克隆或下载本仓库
2. 修改相关配置和内容
3. 部署到Web服务器或GitHub Pages等平台
